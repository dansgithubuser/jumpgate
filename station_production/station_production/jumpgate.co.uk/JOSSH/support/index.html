Bug Reports and Gameplay Issues
If you have encountered a bug that has not seriously affected your character, please post a bug report on the Jumpgate Bug Report Forum with as much detail as you can.

 If your character has been seriously affected by a bug, and you require reimbursement or other corrections in your pilot account, please contact support at pilot_support@netdevil.com. Problems that support can assist with include the following types of issues:
  • Lost items/credits/experience
  • Erroneous pilot stats
  • Missing medals
  • Missions that cannot be completed
 Please include as much information as you can, including specifics on what was lost, pilots involved, and locations and times whenever appropriate. Be sure to include your account name as well.

 Jumpgate Reimbursement Policy
Support will generally restore any losses that are the result of encountering a game bug, after verification is done on pilot records to confirm the loss. Losses due to the actions of other players, lag or network problems, personal computer issues, or related to the sharing of account information are not covered, and cannot be restored.

Billing and Account Problems
For billing and account problems, please email pilot_support@netdevil.com for assistance. Billing and account problems include the following issues:
  • Registration errors
  • Reactivation problems
  • Billing failure
  • Classified accounts
 If you need to edit your billing information or cancel your account, please visit the Edit Account Center. If you are using one of the PayByCash alternative payment options, please contact them directly to make changes to your account information. They can be reached at customerservice@paybycash.com.

Technical Support
For technical support issues, please contact pilot_support@netdevil.com with the following information:
  • Account/Pilot name
  • Detailed description of the problem
  • Any error messages you may have received
  • A copy of your error.txt file
  • A copy of your DxDiag.txt file
 error.txt
This file is located in your Jumpgate install directory, and is erased every time you boot up the game. Please be sure to send a copy that was created immediately after the problem occurred.

 DxDiag.txt
This file is generated by the DirectX Diagnostics program that is installed automatically with DirectX. To access this program, open the Start menu, choose Run, and enter "dxdiag" into the dialog box. Please visit the Display and Sound tabs and run any tests that are available, then choose the 'Save All Information' option to create the DxDiag.txt file.
