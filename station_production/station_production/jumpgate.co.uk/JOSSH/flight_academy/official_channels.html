Official Comm Channels

The following list represents public chat booths and channels officially recognized by TRI. Any communication submitted over these channels should be considered easily susceptible to interception. Therefore, communication should cover only material of an insensitive, non-confidential matter.

Every official booth or channel has an offical purpose. Pilots are mandated to use these channels only for their intended purpose. If a pilot is found to be misusing an official channel, they will be kicked from the system. Misuse is determined at TRI's sole discretion.

To join one of these booths, type:
<f5>:name
where name is replaced with the name of the booth you wish to join.



Channel Purpose
help The default channel all new recruits join automatically. This channel should be used for general help questions and responses.
new This is an alternate help channel, used when there is too much traffic on the default :help channel. This channel should be used for general help questions and responses.
newb This is an alternate help channel, used when there is too much traffic on the default :help channel. This channel should be used for general help questions and responses.
chat This is a general chat and conversations channel. This can be used for all forms of general discussion.
sol This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Solrain faction. This channel should be used for general help questions and responses.
oct This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Octavian faction. This channel should be used for general help questions and responses.
quan This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Quantar faction. This channel should be used for general help questions and responses.
flux This channel is used for intelligence on current Conlux infestation. Communication should be limited to requests for Conflux extermination, information on current Conflux whereabouts, general Conflux hunting strategy, or anything Conflux related.
jobs This channel is used for offering services and hiring assistance for all manner of odd jobs. Chat should cover any jobs being offered by individuals or squads, as well as responses to those offers. This is the channel to join if you are looking for work from other pilots, or if you need to hire another pilot to help you.
roid This channel is used for asteroid farming intelligence. Conversation should surround locations of valuable asteroids, mining tips, and anything asteroid related.
